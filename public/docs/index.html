Getting Started

ratio.css is a geometric ratio-based CSS framework built on IBM Carbon design tokens. Every dimension in the system derives from two variables: a base unit and a scale ratio.

Install

<link rel="stylesheet" href="ratio.css">

The Concept

Every scale step is computed from one formula:

step(n) = base × ration

The default base is 1rem and the default ratio is φ (1.618) — the golden ratio. This produces a harmonious scale where each step is proportionally related to every other step. Spacing, typography, sizing, and component dimensions all reference the same scale.

Scale Tokens

CSS custom properties that define the geometric scale steps, base configuration, and derived values like padding and radius.

Color Tokens

IBM Carbon Gray 100 theme color tokens. All component and utility colors reference these properties.

Spacing Utilities

Padding, margin, and gap classes mapped to scale steps. Level 0 = zero, levels 1-9 map to steps --s-n3 through --s-5.

Typography Utilities

Font size, weight, alignment, line height, and letter spacing classes.

Layout Utilities

Flexbox, grid, display, positioning, border radius, sizing, and miscellaneous utility classes.
